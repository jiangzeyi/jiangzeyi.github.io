1 分钟 · 泽毅

欢迎光顾小破站，小破站目前部署在 Github Page 和 Cloudflare Page 上。

我是江泽一，一介布衣，性格有些内（闷）向（骚）；兴趣爱好寥寥无几，算得上是没有兴趣爱好；没有拿得出手履历和特长；

初中时在网络上初识了 PHP 和 渗透，觉得非常的有趣，正因为这些我现在算是半只脚跨入的计算机相关行业的人了，使用语言主要是 Java。

读书的几年了也建过许多类似的网站，结果不了了之。在建这个站的时候我思前想后，我这样的人能记录些什么东西呢、能给我带来什么等，现在都还没想清楚这一系列的问题。做好一件事最好的时间是十年之前， 其次是现在。本着只管去做时间会给出一切答案的心态创建了这个小破站。
